#EduEgy is an android application that was developed as a messaging service for the Ministry 
of Education in egypt for the internal use of the group of youth working in the ministry.


#Parse server is a platform that allows to host data like usernames,passwords,photos.

-Parse server is stored in own server.

-Can run in any service provider like google cloud and microsoft azure.

#Amazon web service is used as a service provider.

#Ec2 is the service used from AWS.

#Key pair is used to access AWS instances.

#Simple login/sign up system is used to log users to the users activity.

#ListView is used to contain the users.

#Another Listview is used to maintain chat history.
